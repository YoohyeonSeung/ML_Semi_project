꼭 잊지 말아야 할 것은 project interpreter 설정입니다. 

clone을 해준 이후, 
1. File -> setting 클릭
2. 왼쪽 항목에서 Project:ML_Semi_project -> Project Interpreter 클릭
3. 오른쪽에 Project Interpreter : 에서 클릭후 tf_cpu밑에 있는 python을 클릭
(만약 없다면 밑에 show all..을 누르면 나옵니다.)
